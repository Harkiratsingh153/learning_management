# Mini Learning Management System (LMS)

A simple **console-based Learning Management System** built in Python using **Object-Oriented Programming (OOP)** principles.

This project allows administrators to:
- Add and manage courses
- Register students
- Enroll students in courses
- Track lesson completion
- View student progress in each course

Perfect as a beginner/intermediate Python project, school/college assignment, or demonstration of OOP concepts (classes, objects, lists, dictionaries, encapsulation, etc.).

## Features

- Add new courses with a title and number of lessons
- Register new students
- Enroll students into one or more courses
- Record the number of completed lessons for a student in a course
- View detailed progress (completed/total lessons + percentage)
- List all available courses (with enrollment count)
- List all registered students
- Simple text-based menu interface
- Case-insensitive name/title matching

## Technologies Used

- **Python 3.x** (no external libraries required)
- Built-in modules only: `input()`, lists, dictionaries, string methods
